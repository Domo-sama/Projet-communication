Modalités d'inscription


Après avoir choisi le programme qui vous intéresse, vous devez connaître les modalités d'inscription de ces derniers. Les modalités d'inscription diffèrent d'un programme à un autre.

Les modalités pour les programmes réguliers :


Profil Sciences pures :

Des Bulltetins de notes pour les cours de mathématiques, physique ou chimie sont requis afin de pouvoir placer l'élève dans le cours du bon niveau.

Profil Sciences humaines :

Des Bulltetins de notes pour les cours d'histoire, communication ou culture sont requis afin de pouvoir placer l'élève dans le cours du bon niveau.

Profil ouvert :

Aucun bulletin n'est requis pour ce profil.


Les modalités pour les programmes spécialisés :


Profil Sciences et technologie :

Des tests de classement sont nécessaires pour s’inscrire à ce profil.

Profil Sciences humaines et société :

Des tests de classement sont nécessaires pour s’inscrire à ce profil.
